Saat pengguna menekan tombol "Kirim" pada layar pertama (FormData), aplikasi mengirimkan data yang diinput ke layar berikutnya menggunakan Navigator.pushNamed(). Data tersebut dikirim dalam bentuk Map melalui parameter arguments, yang berisi nilai dari setiap field input. Pada layar kedua (TampilData), data diambil menggunakan ModalRoute.of(context)!.settings.arguments dan kemudian ditampilkan.
